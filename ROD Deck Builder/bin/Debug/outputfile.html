        Cards marked with (ε) will gain their skill when evolved.

        RY = Rarity
        R = Realm
        F = Faction
        C = Cost - The amount of Power required to use the card to it's fullest in battle. 50% of this will be consumed.
        MATK = The maximum possible attack value (8-15), MDEF = The maximum possible defense value (8-15).
        AEFF = Attack Efficiency - ATK divided by Cost, rounded off.
        DEFF = Defense Efficiency - DEF divided by Cost, rounded off.
        OEFF = Overall Efficiency - (ATK + DEF) divided by Cost, rounded off.
        ES1 = Primary Event Skill/Specialist (if applicable)
        ES2 = Secondary Event Skill/Specialist (if applicable)

        Skills (Semi-/Permanent)
        • ^ = non-permanent
        • BC = Bands of Crimson (Guild War Points Multiplier 2013) [Obsolete]
        • CH = Chest Hunter (Chest Event Chest drop multiplier)
        • GL = Guiding Light (Chest Point Multiplier)
        • LG = Lunar Gladius (Colosseum Points Multiplier)
        • RA = Raid Event Attack Advantage (Non-Weekly) [Obsolete]
        • RW = Raid Advantage (Weekly)
        • SE = Spearhead Emblem (Guild War Points Multiplier 2014)
        • WB = Wall Breaker (Guild War Wall Attack Advantage)


        Skills (Single Event)
        • CB = Colosseum Boss Advantage (Attack Multiplier) applies ONLY to the Colosseum event that begins immediately after the card was introduced.
        • SA = Spring of Awakening, Chest Point Multiplier for The Spring of Reawakening ONLY
        • SS = Sweet Stalker, Chest Point Multiplier for The Wondrous Luscious Lunacy ONLY
